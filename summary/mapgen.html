Map Generator

This is a javascript project where we create a map that's given an input of how many rooms + complexity (extra connections between randomized rooms). Then it calculates all the different paths between your start room and your desired destination and types out all of the paths in ascending order from shortest to the furtherst.

It's created in HTML, CSS and JS.
